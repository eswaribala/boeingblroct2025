1) Install prerequisites (one-time)
.NET SDK (8 or 9)WiX 
v4 SDK (modern, SDK-style)
dotnet new install WixToolset.Sdk

2) Install the WiX CLI (global .NET tool)
dotnet tool install --global wix
# later, to update:
dotnet tool update --global wix

3)
Name, Manufacturer, Version.UpgradeCode → generate once and keep it forever for this product line:
PowerShell: "[guid]::NewGuid()" → paste it in {…}.
File Source="..\YourApp\publish\YourApp.exe" → match your actual path/name.
  3. Build the MSI
From the Installer folder:
wix build Product.wxs -d PublishDir="..\CustomerApp\publish" -o CustomerApp.msi
You should get YourApp-1.0.0.msi in the same directory.
